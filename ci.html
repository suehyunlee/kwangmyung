시그니처

시그니처는 심볼과 로고타입의 가장 이상적인 조합이므로 규정을 벗어난 조합은 사용할 수 없으며, 적용 환경과 매체에 따라 선택하여 사용할 수 있습니다.

컬러규정

광명산업의 전용색상은 Basic System은 물론 Application System 전반에 걸쳐 사용됩니다. 기본색상은 KMIN Red, KMIN Blue로 하되 적용 매체의 특성상 인쇄되어야 할 경우 아래에 제시된 Color 비율에 따라 표시해야합니다.

KMIN Red

  • HTML : #e60012
  • CMYK : 0, 100, 90, 0
  • RGB : 230, 0, 18
  • PANTONE : 485 C

KMIN Blue

  • HTML : #003969
  • CMYK : 100, 87, 44, 8
  • RGB : 0, 57, 105
  • PANTONE : 654 C

심볼마크 규정

심볼마크는 광명산업의 기업 아이덴티티 시스템의 기본 요소로서 기업의 이미지를 대내외에 표출하는 모든 커뮤니케이션의 핵심이 되는 대표적인 상징입니다.
